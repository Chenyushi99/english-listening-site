# 影视口语听写练习室

一个静态英语听写练习网站，面向影视口语、长难句和不同英语口音训练。

## 功能

- 本地用户名登录，不同用户在同一浏览器里拥有独立进度
- 只播放声音进行英文听写
- 中文提示、首字母提示、答案揭示和难句标记
- 素材按 A2 到 C2 难度分级
- 支持美音、英音、印度英语、日式英语和混合口音分类
- 已学内容测试和错题复习

## 发布到 GitHub Pages

这个项目是纯静态网站，可以直接发布到 GitHub Pages。

1. 把仓库推送到 GitHub。
2. 在 GitHub 仓库页面进入 `Settings` -> `Pages`。
3. `Build and deployment` 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/root`。
5. 保存后等待 GitHub 生成公开访问链接。

## 本地打开

直接用浏览器打开 `index.html` 即可。
